# Smart Inventory Management System

A web-based inventory management system to track stock, manage sales/purchases, and generate reports efficiently for small businesses.

---

## Features

- Real-time tracking of products and stock levels
- Automated low-stock alerts
- Sales, purchase, and transaction management
- Reporting dashboard with analytics
- User-friendly responsive interface

---

## Tech Stack

- **Backend:** PHP, Laravel  
- **Frontend:** Blade Templates / HTML / CSS / JavaScript  
- **Database:** MySQL  
- **Other Tools:** Composer, Artisan

---

## Prerequisites

- PHP >= 8.0
- Composer
- MySQL
- Web server (optional: XAMPP, Laragon)

---

## Installation

1. Clone the repository:  
   ```bash
   git clone <your-repo-link>
   cd SmartInventoryManagementSystem

2. Install dependencies:
   ```bash
   composer install
3. Configure the environment:
   ```bash
    cp .env.example .env
    php artisan key:generate

4. Run migrations and seed the database:
    ```bash
   php artisan migrate --seed

5. Start the development server:
   ```bash
    php artisan serve
